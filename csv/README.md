# CSV storage

--- 

- Here is the directory that contains `csv` files with the information about articles (original and preprocessed)
- Directory `csv` is necessary for **SciTriveo** running

## Basic files
- `abstracts_lemmatized.csv` – preprocessed file with the data about articles with the research abstract and metadata
- `abstracts.csv` *(optional)* – raw dataset with research abstracts (needed if you want to preprocess it on your own)
